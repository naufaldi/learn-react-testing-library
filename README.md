**Test:** React Testing Library and Jest

## Run Locally

Clone the project

```bash
  git clone https://github.com/naufaldi/learn-react-testing-library.git
```

Go to the project directory

```bash
  cd learn-react-testing-library
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run devStart
```
